# Quidax

A ruby gem for easy integration of [Quidax](https://docs.quidax.com/docs/getting-started)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'quidax'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install quidax
